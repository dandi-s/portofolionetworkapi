# Network Operations Integration API

Production-ready API for ISP network operations management.

## Quick Start
```bash
# Install dependencies
go mod download

# Start databases
docker compose up -d

# Run API
go run src/main.go
```

## Features
- Device management
- Bulk configuration
- Monitoring integration
- Incident tracking

## Tech Stack
- Go 1.21+
- PostgreSQL 15
- Redis 7
- Docker
